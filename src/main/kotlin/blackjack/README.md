# Card
- [x] 네 가지 종류의 모양과 1 ~ 10, J, Q, K 카드가 있다.
- [x] 정해진 카드만 생성할 수 있다.

# Deck
- [x] 카드 52장을 가지고 있다.
- [x] 카드를 섞을 수 있다.
- [x] 카드를 한 장씩 뽑을 수 있다.

# Player
- [x] 이름을 가진다.
- [x] 카드를 받을 수 있다.
- [x] 현재 가진 카드를 저장한다.
- [x] 21점을 초과하여 카드를 받을 수 없다.

# Cards
- [x] 현재 카드 점수를 계산한다.

# Dealer
- [x] 현재 카드 점수가 16점 이하면 추가로 카드를 받는다.
- [x] 현재 카드 점수가 17점 이상이면 추가로 카드를 받을 수 없다.
